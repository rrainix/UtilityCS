# C# Utility Library
A wide-range `Net 9.0` C# Utility Library combining all of my projects code

## Features
- Serialization and Deserialization (JSON, XML, Binary)
- File Management
- Collections
- Native Collections
- Randomization
- Cryptography
- Password Generation
- Encryption and Decryption (AES)
- Mathematics
- Logging
- Mini LINQ
- CMD Utils

## How to install

### Option 1 - Clone the Repository
1. Clone with  
   `https://github.com/Ben-Scr/UtilityCS`
2. Add `UtilityCS.csproj` to your project  
   **or** test the features directly in `UtilityPlayground.csproj`.

### Option 2 - Install via NuGet
1. Install the package from NuGet:
   ```bash
   dotnet add package UtilityCS
